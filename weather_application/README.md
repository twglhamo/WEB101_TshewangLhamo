# Weather API Dashboard 

## 📌Overview
This project is a simple Weather API Dashboard that allows users to:
- Get current weather using a city name (GET request)
- Save locations (POST request)
- Update saved locations (PUT request)
- Delete saved locations (DELETE request)

It demonstrates how REST APIs work in a real web application.

---

##  Features

-  Search weather by city name
-  View temperature, humidity, wind, and conditions
-  Save locations using a POST request
-  Edit saved locations using PUT request
-  Delete saved locations using DELETE request
-  Displays API response details

---

##  Technologies Used

- HTML
- CSS
- JavaScript (Vanilla JS)
- OpenWeatherMap API
- JSONPlaceholder API

---

## How to Run the Project

1. Download or clone the project
2. Open the folder in VS Code
3. Open `index.html` in your browser
4. Enter a city name and click **Get Weather**
5. Use other tabs to test POST, PUT, and DELETE operations


##  API Endpoints Used

- GET: `https://api.openweathermap.org/data/2.5/weather`
- POST: `https://jsonplaceholder.typicode.com/posts`
- PUT: `https://jsonplaceholder.typicode.com/posts/{id}`
- DELETE: `https://jsonplaceholder.typicode.com/posts/{id}`
